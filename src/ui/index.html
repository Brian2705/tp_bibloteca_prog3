Multa calculada (1% por día, máx 30 días)

Implementación Java (BibliotecaService)
public double calcularMulta(int dias, double valor) {
    if (dias <= 0) return 0;
    if (dias > 30) dias = 30;
    return valor * 0.01 + calcularMulta(dias - 1, valor);
}
